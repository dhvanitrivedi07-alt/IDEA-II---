# SmartTask Hub 🧠✅

SmartTask Hub is a modern task management application designed to help users organize, prioritize, and manage their daily tasks efficiently with a clean and intuitive interface.

---

## ✨ Features

- Create, edit, and delete tasks
- Assign **importance using text labels** (Low / Medium / High)
- Manage task priorities clearly and easily
- User-friendly and responsive UI
- Clean design for better productivity

---

## 🛠️ Tech Stack

This project is built using modern frontend technologies:

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn-ui
